Description
A simple web application built as part of the freeCodeCamp JavaScript Certification. The app allows users to create, view, and delete bookmarks organized by categories, using the browser's local storage for data persistence.
🚀 Features
Data Persistence: Saves bookmarks in localStorage so they remain after page refresh.
Category Filtering: View bookmarks specifically by their assigned category.
Error Handling: Robust JSON parsing using try...catch blocks to prevent app crashes from corrupted data.
DOM Manipulation: Dynamic HTML generation based on user input.
🛠 Tech Stack
HTML5 (Semantic structure)
CSS3 (Layout and hidden states)
JavaScript (ES6+, DOM API, LocalStorage)
🧠 What I Learned
How to manage application state using global arrays synchronized with localStorage.
Implementing defensive programming by validating data types and handling exceptions.
Handling complex user interactions like radio button selection for deletion logic.
Solving synchronization issues between the UI and the data layer.
🔧 How to Run
Clone the repository.
Open index.html in any modern web browser.
